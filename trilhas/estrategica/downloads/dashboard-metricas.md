## Métricas Fundamentais

### 1. GMV (Gross Merchandise Value)
- **O que é:** Volume total de vendas
- **Benchmark:** R$50K-100K/mês para considerar escala
- **Fórmula:** GMV = Tráfego × CVR × AOV

### 2. CVR (Conversion Rate)
- **O que é:** Taxa de conversão
- **Benchmark:** 2-5% média do TikTok Shop
- **Meta:** Acima de 3%

### 3. AOV (Average Order Value)
- **O que é:** Ticket médio
- **Benchmark:** R$50-80
- **Otimização:** Bundles, upsell

### 4. ROAS (Return on Ad Spend)
- **O que é:** Retorno sobre investimento em ads
- **Benchmark:** 3x-5x mínimo para escalar
- **Fórmula:** Receita / Custo de Ads

## Rotinas de Análise

### Diário (10 min)
- [ ] GMV do dia anterior
- [ ] Número de pedidos
- [ ] Vídeos mais performantes (24h)
- [ ] Alertas de estoque
- [ ] Reviews negativos

### Semanal (30 min)
- [ ] Comparativo GMV semana vs anterior
- [ ] Top 5 vídeos da semana
- [ ] Performance por afiliado
- [ ] AOV e ticket médio
- [ ] Taxa de retorno/cancelamento

### Mensal (1h)
- [ ] GMV total e crescimento %
- [ ] Margem líquida real
- [ ] Produtos mais vendidos
- [ ] ROI por canal
- [ ] Performance Pareto de afiliados
- [ ] Projeção próximo mês

## Diagnóstico de Problemas

### Views baixas
- [ ] Watch time baixo → Melhorar primeiros 3s
- [ ] Horário errado → Testar diferentes horários
- [ ] Conta shadowbanned → Verificar violações

### Poucos cliques no produto
- [ ] CTA fraco → Adicionar call-to-action
- [ ] Produto não mostrado → Destacar visualmente
- [ ] Audiência errada → Ajustar conteúdo

### Add-to-Cart baixo
- [ ] Preço alto → Testar preços
- [ ] Fotos ruins → Melhorar imagens
- [ ] Frete caro → Considerar frete grátis

### Abandono de carrinho
- [ ] Checkout complexo → Simplificar
- [ ] Frete surpresa → Ser transparente
- [ ] Falta de confiança → Adicionar reviews
