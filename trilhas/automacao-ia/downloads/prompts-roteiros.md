## Prompt Base: Roteiro de Vídeo TikTok

```
Você é um copywriter especializado em TikTok Shop que cria roteiros virais para [NICHO].

PRODUTO: [Nome do produto]
BENEFÍCIO PRINCIPAL: [Benefício]
PÚBLICO-ALVO: [Descrição do público]
DURAÇÃO: [15/30/60 segundos]

Crie um roteiro seguindo esta estrutura:

1. GANCHO (0-3s): Frase impactante que prende atenção
2. PROBLEMA (3-8s): Dor do público que o produto resolve
3. SOLUÇÃO (8-20s): Apresentação do produto como solução
4. PROVA (20-25s): Resultado/benefício tangível
5. CTA (25-30s): Chamada para ação natural

REGRAS:
- Tom informal e conversacional
- Máximo 150 palavras
- Sem "clique no link" ou "compre agora"
- Use linguagem do público-alvo
- Inclua emoji relevante no gancho
```

## Prompt: Variações de Gancho

```
Crie 10 ganchos diferentes para vídeo de TikTok sobre [PRODUTO].

Use estes formatos:
- 2 perguntas intrigantes (começam com "Você sabia que...")
- 2 com elemento de choque/surpresa
- 2 revelando um "segredo"
- 2 com promessa de resultado rápido
- 2 com tom levemente polêmico

REGRAS:
- Máximo 10 palavras cada
- Tom informal e direto
- Criar curiosidade imediata
- Funcionar sem contexto
```

---

## Prompt: Variações de CTA

```
Crie 10 CTAs (chamadas para ação) para vídeo de TikTok Shop.

PRODUTO: [Nome do produto]
OBJETIVO: Fazer o espectador tocar no carrinho amarelo

Tipos de CTA:
- 2 com urgência
- 2 com escassez
- 2 com prova social
- 2 focados em benefício
- 2 que pedem engajamento (comentário)

REGRAS:
- Máximo 15 palavras cada
- Soar natural, não vendedor
- Evitar: "link na bio", "compre agora", "acesse o site"
- Preferir: "carrinho amarelo", "salva pra depois", "comenta"
```

## Prompt: Roteiro Problema-Solução

```
Crie um roteiro de [30] segundos no formato PROBLEMA-SOLUÇÃO.

PRODUTO: [Nome]
PROBLEMA que resolve: [Descreva a dor]
PÚBLICO: [Quem sofre com isso]

Estrutura:
1. Gancho que identifica o problema (3s)
2. Amplifique a dor - mostre consequências (5s)
3. Apresente a solução como descoberta (10s)
4. Demonstre resultado/transformação (7s)
5. CTA natural (5s)

O espectador deve pensar: "Isso sou eu!"
```

## Prompt: Roteiro Tutorial

```
Crie um roteiro de tutorial para TikTok sobre como usar [PRODUTO].

PRODUTO: [Nome]
RESULTADO FINAL: [O que a pessoa consegue fazer]
DURAÇÃO: [60 segundos]

Estrutura:
1. Gancho: "Como [resultado] em [tempo]" (3s)
2. Passo 1 - [Ação simples] (10s)
3. Passo 2 - [Ação simples] (10s)
4. Passo 3 - [Ação simples] (10s)
5. Resultado final visível (10s)
6. CTA: "Você consegue!" + carrinho (7s)

REGRAS:
- Cada passo deve ser visual e claro
- Use palavras simples
- Mostre, não apenas fale
```

## Prompt: Roteiro Antes/Depois

```
Crie um roteiro de transformação ANTES/DEPOIS para [PRODUTO].

PROBLEMA INICIAL (Antes): [Situação ruim]
RESULTADO FINAL (Depois): [Situação desejada]
TEMPO de transformação: [Período]

Estrutura:
1. Gancho visual do ANTES (3s)
2. "Eu estava assim..." - contexto (5s)
3. "Então descobri..." - apresenta produto (8s)
4. Transição dramática (2s)
5. DEPOIS impressionante (7s)
6. CTA de transformação (5s)

O contraste deve ser visualmente impactante.
```

## Prompt: Roteiro Storytime

```
Crie um roteiro de storytime sobre experiência com [PRODUTO].

NARRATIVA: [Resuma a história em 1 linha]
EMOÇÃO PRINCIPAL: [alegria/surpresa/alívio]
PRODUTO: [Nome]

Estrutura:
1. Gancho narrativo: "Vocês não vão acreditar..." (3s)
2. Contexto da situação (8s)
3. O problema/desafio (7s)
4. A descoberta do produto (5s)
5. A resolução/resultado (5s)
6. Conclusão + CTA (2s)

REGRAS:
- Primeira pessoa
- Emoção genuína
- Detalhes específicos que geram conexão
```

## Prompt: Listicle/Top X

```
Crie um roteiro de "Top 5 [CATEGORIA]" incluindo [PRODUTO].

CATEGORIA: [Ex: produtos de skincare]
PRODUTO DESTACADO: [Seu produto - deve ser o #1]
OUTROS PRODUTOS: [Liste 4 complementares]

Estrutura:
- Gancho: "5 [produtos] que mudaram minha vida" (3s)
- #5: [Produto] - benefício rápido (5s)
- #4: [Produto] - benefício rápido (5s)
- #3: [Produto] - benefício rápido (5s)
- #2: [Produto] - benefício rápido (5s)
- #1: [SEU PRODUTO] - destaque maior (10s)
- CTA: "O #1 está no carrinho" (2s)

O #1 deve ter o maior destaque e tempo.
```

## Prompt: Geração em Lote

```
Com base nesta IDEIA BASE, gere 5 variações completas de roteiro:

IDEIA BASE:
- Produto: [Nome]
- Benefício: [Principal vantagem]
- Público: [Quem é]
- Duração: 30 segundos cada

Gere 5 roteiros, cada um com abordagem diferente:
1. Formato PROBLEMA-SOLUÇÃO
2. Formato TUTORIAL rápido
3. Formato ANTES/DEPOIS
4. Formato STORYTIME pessoal
5. Formato REVIEW honesto

Cada roteiro deve ter:
- Gancho único
- CTA diferente
- Tom ligeiramente diferente
- Mesma mensagem central

Numere claramente cada variação.
```

## Prompt: Análise de Viral

```
Analise este vídeo viral de TikTok e extraia os elementos de sucesso:

[COLE A TRANSCRIÇÃO OU DESCRIÇÃO DO VÍDEO]

Identifique:
1. Tipo de gancho usado
2. Estrutura narrativa
3. Elementos emocionais
4. Técnicas de retenção
5. Tipo de CTA
6. Por que funcionou (hipóteses)

Com base nessa análise, crie 3 variações de roteiro para [MEU PRODUTO] usando os mesmos elementos.
```

## Dicas de Uso

### Temperatura da IA
- **Ganchos criativos:** Temperature 0.8-0.9
- **Roteiros estruturados:** Temperature 0.6-0.7
- **CTAs diretos:** Temperature 0.4-0.5

### Versionamento
- Salve prompts que funcionam
- Documente alterações
- Teste variações pequenas
- Meça resultados por versão

### Iteração
1. Gere 5-10 variações
2. Selecione as 3 melhores
3. Teste no TikTok
4. Analise performance
5. Ajuste o prompt baseado em dados
