# Checklist de Seguranca TikTok
## TikTok Shop Academy 2025 - Trilha Automacao IA

---

## O que e Permitido pelo TikTok

### Praticas Seguras (Zona Verde)

- [ ] Usar TikTok Studio para agendamento
- [ ] Publicar 3-5 videos por dia
- [ ] Usar hashtags relevantes ao nicho
- [ ] Responder comentarios genuinamente
- [ ] Fazer lives periodicamente
- [ ] Usar sons trending da biblioteca oficial
- [ ] Linkar produtos do TikTok Shop no carrinho
- [ ] Usar ferramentas oficiais de analytics
- [ ] Crescer seguidores organicamente
- [ ] Colaborar com criadores via sistema oficial

---

## Zona Cinza (Cuidado)

### Praticas que Requerem Cautela

- [ ] Agendamento via ferramentas terceiras (Botato, Later)
  - *Risco: Baixo se usado com moderacao*
  - *Limite: Nao ultrapassar 10 posts/dia*

- [ ] Multiplas contas no mesmo dispositivo
  - *Risco: Medio*
  - *Mitigacao: Use dispositivos/IPs diferentes*

- [ ] Repostar conteudo proprio
  - *Risco: Baixo a medio*
  - *Regra: Minimo 2 semanas de intervalo*

- [ ] Usar IA para geracao de conteudo
  - *Risco: Baixo*
  - *Regra: Nao mencione que e gerado por IA*

- [ ] Videos com avatar de IA
  - *Risco: Baixo a medio*
  - *Regra: Use como parte do conteudo, nao 100%*

- [ ] Automacao de DMs
  - *Risco: Medio a alto*
  - *Limite: Max 20-30 DMs personalizadas/dia*

---

## O que Pode Gerar Banimento (Zona Vermelha)

### Praticas Proibidas - NUNCA Fazer

- [ ] Comprar seguidores/likes/views
- [ ] Usar bots para engajamento automatico
- [ ] Mesmo video exato em multiplas contas
- [ ] Spam de links externos
- [ ] Conteudo copiado de outros criadores
- [ ] Hashtags banidas ou irrelevantes em massa
- [ ] Follow/unfollow massivo
- [ ] Criar contas falsas em serie
- [ ] Evitar deteccao de forma explicita
- [ ] Manipular metricas artificialmente

---

## Sinais de Shadowban

### Como Identificar

- [ ] Views caem 80%+ repentinamente
- [ ] Videos nao aparecem no For You
- [ ] Hashtags nao indexam
- [ ] Comentarios somem
- [ ] Conta nao aparece em buscas
- [ ] Notificacoes de "revisao" frequentes

### Acoes se Shadowban Detectado

1. [ ] Parar de postar por 24-48h
2. [ ] Deletar videos com violacoes
3. [ ] Remover hashtags problematicas
4. [ ] Verificar se conta foi restrita
5. [ ] Voltar com 1 video/dia por 1 semana
6. [ ] Engajar organicamente (comentar, curtir)
7. [ ] Se nao recuperar em 2 semanas, focar em outra conta

---

## Regras para Multiplas Contas

### Setup Seguro

- [ ] Cada conta em dispositivo diferente
- [ ] IPs diferentes (proxies residenciais se necessario)
- [ ] Emails diferentes e verificados
- [ ] Numeros de telefone diferentes
- [ ] Nao logar em mais de 1 conta por sessao
- [ ] Comportamento humano em cada conta

### Limites por Conta

| Acao | Conta Nova (0-30 dias) | Conta Estabelecida |
|------|------------------------|-------------------|
| Posts/dia | 3-5 | 5-10 |
| Comentarios/dia | 20-30 | 50-100 |
| Follows/dia | 50-100 | 100-200 |
| DMs/dia | 10-20 | 20-50 |
| Intervalo entre posts | 3-4h | 2-3h |

---

## Checklist Pre-Publicacao

### Antes de Publicar Qualquer Video

- [ ] Video e original ou suficientemente editado
- [ ] Nao contem musica com copyright restrito
- [ ] Hashtags sao relevantes (max 5-7)
- [ ] Descricao nao tem links externos suspeitos
- [ ] Produto linkado esta ativo no TikTok Shop
- [ ] Nao viola diretrizes de comunidade
- [ ] Nao faz claims medicos/financeiros proibidos
- [ ] CTA nao e spam ou enganoso

---

## Checklist de Conta Saudavel

### Verificacao Semanal

- [ ] Views consistentes com historico
- [ ] Engajamento estavel ou crescendo
- [ ] Nenhuma notificacao de violacao
- [ ] Lives funcionando normalmente
- [ ] Comentarios aparecendo para outros
- [ ] Hashtags indexando corretamente
- [ ] TikTok Shop funcionando sem restricoes
- [ ] Analytics atualizando normalmente

---

## Plano de Contingencia

### Se Conta for Banida

1. [ ] Nao criar nova conta imediatamente
2. [ ] Analisar motivo do ban
3. [ ] Apelar se ban injusto (1x apenas)
4. [ ] Aguardar resposta por 7-14 dias
5. [ ] Se negado, focar em contas backup
6. [ ] Nova conta: dispositivo novo, IP novo
7. [ ] Crescer nova conta organicamente primeiro

### Contas Backup (Recomendado)

- [ ] Ter 2-3 contas secundarias em warmup
- [ ] Postar 1x/dia para manter ativas
- [ ] Nao usar para promocao ate estabelecidas
- [ ] Manter engagement organico basico

---

## Automacao Responsavel

### Limites Recomendados

```
Publicacao automatica: Max 10 videos/dia total
Intervalo minimo: 2 horas entre posts
Repost: Max 30% do conteudo
Variacao: Nunca mesmo video em 2 contas no mesmo dia
Hashtags: Rotacionar, nao repetir exatamente
Captions: Variar, mesmo para produto similar
```

### Ferramentas Aprovadas (Uso Moderado)

- TikTok Studio (oficial)
- Botato
- Later
- Buffer
- Hootsuite

### Ferramentas de Risco (Usar com Extremo Cuidado)

- Qualquer bot de engajamento
- Ferramentas de follow automatico
- Scrapers de dados
- Automacao de DM em massa
