Orquestracao com n8n e Make

O cerebro da automacao - conectando todas as ferramentas

4 Topicos

  • n8n: Open-source, self-hosted, mais controle tecnico. Ideal para quem quer customizacao total e nao quer pagar por execucao.

    Make: Cloud-based, interface visual amigavel, integracao nativa com +1000 apps. Ideal para comecar rapido.

    Recomendacao: Comece com Make para validar fluxos, migre para n8n quando escalar para reduzir custos.

  • O que sao: Webhooks sao "gatilhos" que disparam automacoes quando algo acontece (nova linha no Sheets, novo registro no Notion, etc).

    Exemplos: Nova ideia no Notion â†’ Gera roteiro automaticamente. Novo video pronto â†’ Publica no TikTok.

    Dica: Configure webhooks para cada etapa do funil de conteudo para automacao completa.

  • Setup: Crie conta na OpenAI/Anthropic â†’ Gere API Key â†’ Configure no n8n/Make como "HTTP Request" ou modulo nativo.

    Custo medio: GPT-4: ~$0.03 por roteiro. Claude: ~$0.02 por roteiro. Escala de 1000 roteiros/mes = ~$20-30.

    Dica Pro: Use GPT-3.5 para tarefas simples e GPT-4/Claude apenas para roteiros finais.

  • O que e: Processar multiplos itens de uma vez. Ex: 50 ideias â†’ 50 roteiros em uma unica execucao.

    Como fazer: Use "Split in Batches" no n8n ou "Iterator" no Make para processar listas.

    Cuidado: Respeite rate limits das APIs. Configure delays de 1-2s entre requisicoes.

Crie videos "humanizados" em escala sem gravar

4 Topicos

    Vantagens: Avatar fixo = consistencia de marca. Voz clonada = sua voz sem gravar. Escala infinita.

    Custo: ~$1-2 por video de 30s. Plano Creator: $29/mes (~15 videos).

  • O que faz: Transforma videos longos em shorts automaticamente. IA identifica os melhores momentos.

    Uso ideal: 1 video de 10min â†’ 20 shorts de 30-60s. Reaproveitamento infinito de conteudo.

    Dica: Grave 1 video longo por semana, deixe Opus gerar dezenas de cortes.

  • Como funciona: Grave 2-5 minutos da sua voz. IA aprende seu tom, ritmo e sotaque. Gera audios novos automaticamente.

    Ferramentas: ElevenLabs (melhor qualidade), HeyGen (integrado), Play.ht (mais barato).

    Etica: Use apenas sua propria voz. Nunca clone voz de terceiros sem permissao.

  • Passo 1: Roteiro pronto (do Modulo 2) â†’ Envia para API do HeyGen.

    Passo 2: HeyGen renderiza video â†’ Webhook avisa quando pronto.

05

Publicacao Automatica (Botato)

Agendamento em massa e rotacao de contas

4 Topicos

  • Botato: Publicacao automatica no TikTok via automacao. Agenda, posta e monitora.

    Alternativas: Metricool (oficial), Later, Publer. Cada um com pros/contras.

    Importante: TikTok tem API limitada. Ferramentas nao-oficiais tem risco de restricao.

  • Conceito: Suba 50 videos de uma vez â†’ Configure horarios otimos â†’ Sistema posta automaticamente.

    Horarios otimos: 7h, 12h, 18h, 21h (horarios de pico Brasil). Varie para descobrir o melhor.

    Frequencia: 3-5 videos/dia por conta e o sweet spot. Mais que isso pode parecer spam.

  • Estrategia: Multiplas contas em nichos diferentes. Se uma cai, as outras continuam.

    Organizacao: Conta A = Beleza, Conta B = Casa, Conta C = Tech. Conteudo especifico para cada.

    Cuidado: Nao use o mesmo dispositivo/IP para todas. TikTok detecta e pode restringir.

  • Sinais de shadowban: Views caem drasticamente, videos nao aparecem na FYP, 0 views em videos novos.

    Prevencao: Varie conteudo, nao poste identico em multiplas contas, use hashtags diferentes.

    Recuperacao: Pause 48-72h, poste conteudo organico, interaja manualmente, volte gradualmente.
